#  **Project Title: Cashier Checkout Simulator**

  **Project Overview:**

  Cashier Checkout Simulator is a graphical application designed to help staff/shop owners manage transactions easily, It allowes users to add, remove and edit items from their shopping list calculating total amount to be paid finally generating a receipt.                             
  **Features:**

  1. Item Entry using name, quantity and price.
  2. Edit item information from the list.
  3. Remove item from the list.
  4. Generate the final receipt for the order.
  5. Whole Graphic UI system is made through Tkinter.

  **Technologies/Tools Used:**

  1. Python 3.13.7
  2. Python Datetime Module
  3. Python Tkinter Module

  **Steps to install and run the project:**
  
  1. Python 3.13.7 should be downloaded.
  2. Tkinter module should be downloaded.
  3. Run the main Python file.

  **Instructions for testing:**

  1. Run the program.
  2. Provide the Item Entry using "Add Item".
  3. Select an item in the shopping list an click "Edit Item" to edit its information.
  4. Select an item in the shopping list an click "Remove Item" to remove it.
  5. Click "Generate Receipt" to generate the final receipt.
  6. Click "Checkout" to finish the order and clear the list.
